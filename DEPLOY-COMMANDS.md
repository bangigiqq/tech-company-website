# Website Deployment - 部署命令
# 作者：Silver ✨

# ============================================
# 呢啲係你創建 GitHub 倉庫後需要運行嘅命令
# ============================================

# 進入網站目錄
cd C:\Users\ChrisL\.openclaw\workspace\website

# 添加 remote（呢個命令要喺你創建倉庫之後先用）
git remote add origin https://github.com/YOUR_USERNAME/tech-company-website.git

# 推送到 GitHub
git push -u origin main

# ============================================
# 啟用 GitHub Pages
# ============================================

# 1. 去 https://github.com/settings/repositories
# 2. 點擊 "tech-company-website" 倉庫
# 3. 點擊左側 "Pages"
# 4. Source 選擇 "main" branch
# 5. 點擊 Save

# ============================================
# 完成！
# 網址會係：https://YOUR_USERNAME.github.io/tech-company-website/
# ============================================
